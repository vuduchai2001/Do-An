# Web workspace

Đây là frontend React + Vite chính của monorepo.

## Vai trò

- cung cấp web UI cho hệ thống mới
- đóng vai trò control plane cho backend
- chỉ giao tiếp với backend qua public API contracts

## Ghi chú

Workspace này hiện được scaffold lại từ template React + TypeScript + Vite chuẩn để giữ baseline sạch, dễ nâng cấp, và dễ mở rộng về sau.
